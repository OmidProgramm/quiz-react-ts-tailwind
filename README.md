# TypeScript Quiz App

This is a simple Quiz application built with **React**, **TypeScript**, and **Tailwind CSS**.  
The app consists of **10 multiple-choice questions** about TypeScript and demonstrates basic React concepts such as state management, event handling, and conditional rendering.

## Live Demo

You can try the quiz online here: [Live Demo](https://omidprogramm.github.io/quiz-react-ts-tailwind/)

## Features

- Interactive quiz with question navigation
- Score calculation based on correct answers
- Dynamic feedback: correct answers highlighted in green, wrong answers in red
- Option to restart the quiz and try again
- Fully responsive design using Tailwind CSS

## Technologies Used

- React
- TypeScript
- Tailwind CSS
